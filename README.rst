==============
nethserver-php
==============

Database example: ::

 php=configuration
    DateTimezone=Europe/Rome
    ExposePhp=
    MaxExecutionTime30=
    MemoryLimit=32
    PostMaxSize=
    ShortOpenTag=
    UploadMaxFilesize=
    MaxFileUploads=

MemoryLimit, PostMaxSize, UploadMaxFilesize are expressed in MB units.
MaxFileUploads is set by default to 20
